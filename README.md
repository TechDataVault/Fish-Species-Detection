# iMATSYA Marine Fish Detection System

## Introduction
The iMATSYA system is designed to detect marine fish species using advanced image processing and machine learning techniques. It aims to assist in marine biodiversity research and fisheries management.

## Features
- **Image Classification**: Classifies images of marine fish into species categories.
- **User-Friendly Interface**: Easy to use interface for researchers and practitioners.
- **Real-Time Detection**: Capable of detecting fish in real-time through live video feeds.

## Installation
To install iMATSYA, clone the repository and install the required dependencies:
```bash
git clone https://github.com/TechDataVault/Fish-Species-Detection.git
cd Fish-Species-Detection
pip install -r requirements.txt
```

## Usage
After installation, use the following command to run the detection system:
```bash
python detect.py --input <input_file> --output <output_file>
```
